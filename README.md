## Description
Eye Tracking System is a computer vision application that tracks eye movements, gaze direction, and blinks in real time using a webcam. It uses facial landmark detection to identify eye regions, enabling hands-free interaction and attention monitoring.

## Features
- Real-time eye gaze tracking
- Blink detection
- Facial landmark-based eye localization
- Hands-free interaction
- Webcam-based implementation

## Technologies Used
- Python
- OpenCV
- Dlib
- Machine Learning
